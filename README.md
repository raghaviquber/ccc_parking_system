Smart Parking System 🚗
📌 Description
This is a coding skills project developed to demonstrate problem-solving and algorithmic thinking using a Smart Parking System. The application allocates parking slots efficiently and compares different algorithmic approaches to find optimal solutions.

🎯 Purpose
This project was built to practice:

Data Structures & Algorithms
Problem-solving techniques
Full-stack development
API integration between frontend and backend

🛠 Tech Stack
Frontend: React (Vite + TypeScript)
Backend: Python (Flask)
Algorithms: Greedy & Dynamic Programming

📁 Project Structure
backend/ → Flask backend (API)
smart-parking-frontend/ → React frontend

⚙️ Features
Allocate parking slots efficiently
Compare Greedy vs Dynamic Programming algorithms
Display distance and time taken
Interactive and user-friendly UI
Real-time communication between frontend and backend

🔗 API Endpoints
POST /api/park → Allocate parking slot
POST /api/compare → Compare algorithms
▶️ How to Run Locally
1. Clone the repository
git clone https://github.com/your-username/smart-parking-system

2. Run Backend
cd backend pip install -r requirements.txt py app.py

Backend runs on: http://127.0.0.1:10000

3. Run Frontend
cd smart-parking-frontend npm install npm run dev

Frontend runs on: http://localhost:8080
